Eight Queens
============

Write a program that will generate all solutions to the Eight queen puzzle[0].
Do not cheat and write your own version. Read only the introduction and the
history section on wikipedia.  After having a working solution, consult the
wikipedia page for possible optimizations.  If you decide to do optimizations,
measure the time so you know how much speedup you got.

[0] https://en.wikipedia.org/wiki/Eight_queens_puzzle
